🚀 项目展示

Redis 集群监控平台

该平台基于 Lua 与 OpenResty 构建，提供简洁直观的界面，实时展示 Redis 集群的运行状态、性能指标和慢查询详情。通过精准的数据分析与告警机制，帮助运维快速定位性能瓶颈与故障，提高集群稳定性与响应速度。项目注重轻量化设计，易于部署与扩展，是高效管理 Redis 生态的重要利器。

FOLIO 一键部署脚本

此脚本专为快速搭建 FOLIO 多模块环境设计，实现了自动化配置 okapi 网关、Nginx 反向代理及模块启用流程。大幅简化部署步骤，提升开发与测试效率，支持灵活定制与扩展。适合团队快速搭建完整的图书馆管理系统环境，加速项目上线和迭代。

Lua API 网关插件

开发了一套基于 OpenResty 的 Lua API 网关插件，支持身份验证、请求限流、日志记录及动态路由等功能。通过灵活配置，实现对微服务的统一管理与安全防护，大幅提升系统的可维护性和扩展能力。

服务器自动化运维脚本

编写多套 Shell 和 Python 自动化脚本，实现服务器巡检、日志收集与故障报警自动化。通过定时任务与邮件通知，大幅提升运维效率，降低人工误操作风险，确保服务器稳定高效运行。

个人技术博客平台

基于静态网站生成器搭建个人博客，集成 Markdown 支持、代码高亮及评论系统。分享开发经验与技术心得，助力持续学习与成长，同时实现简洁美观的阅读体验。
